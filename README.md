# Digikala Sentiment Analysis

This project performs sentiment analysis on Persian Digikala product reviews using NLP techniques and machine learning models.

## Features
- Text preprocessing: normalization, tokenization, stopword removal, stemming
- Sentiment labeling: Positive/Negative (excluding neutral reviews)
- Feature extraction: TF-IDF with n-grams
- Classification: Logistic Regression and Naive Bayes
- Evaluation metrics: Accuracy, Precision, Recall, F1-Score
- Visualization: Top TF-IDF words per sentiment class

## Dataset
Persian product reviews scraped from Digikala.
